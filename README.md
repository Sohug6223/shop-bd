# Shop BD – Bangladeshi E-Commerce (React + Vite)

A beautiful, fully functional e-commerce website built with pure React.  
mobile responsive, localStorage powered.

## Features
- Product grid with live search
- Add to cart + quantity update
- Checkout page with order summary & tax
- Orders history page
- "Added to cart" animation
- 100+ real products with images
- No backend needed – everything works offline

## Live Demo
https://banglaecommerce.netlify.app/

## Tech Stack
- React 18 + Vite
- React Router v6
- LocalStorage persistence
- Pure CSS (no framework)
